# Keel tokens

This directory contains generated token artifacts exported from `colors_and_type.css`.

Generated files:
- `tokens/tokens.json` — categorized token values for tooling.
- `tokens/tokens.css` — distributable `:root` CSS variable bundle.
- `tokens/tokens.ts` — typed constant export for TypeScript consumers.

Commands (run from repo root):
- `npm run tokens:build`
- `npm run tokens:check`
