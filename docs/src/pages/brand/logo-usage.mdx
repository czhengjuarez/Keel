The Ops Forward brand uses two approved logo files for documentation and product surfaces.

## Logo files

| File | Format | Usage |
|---|---|---|
| `logo1.svg` | SVG | Primary logo for light and neutral surfaces |
| `Logo2.svg` | SVG | Alternate logo for dark or constrained treatments |

## Clear space

Maintain adequate breathing room around the logo. Keep at least one mark-height of clear space on all sides.

## Minimum size

| Asset | Minimum size |
|---|---|
| `logo1.svg` | 120px width |
| `Logo2.svg` | 120px width |

## Color variations

## Placement

- **Header:** Use `logo1.svg` at a readable fixed height
- **Dark hero sections:** Use `Logo2.svg` for stronger contrast
- **Social profiles:** Prefer a square-cropped export from the approved source logo
- **Email signature:** Use `logo1.svg` at 120px+ width
