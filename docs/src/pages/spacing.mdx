## Border radii

Used for rounding corners on buttons, cards, and containers.

| Token | Value | Usage |
|---|---|---|
| `--of-radius-xs` | 4px | Inline chips, small tags |
| `--of-radius-sm` | 6px | Inputs, small buttons |
| `--of-radius-md` | 10px | Buttons (default) |
| `--of-radius-lg` | 14px | Cards |
| `--of-radius-xl` | 20px | Panels, modals |
| `--of-radius-2xl` | 28px | Large surfaces |
| `--of-radius-pill` | 999px | Badges, avatars |
