Use banners for page-level messages that should remain visible until dismissed or resolved.

## Example

<div class="demo demo__stack" style="align-items: stretch;">
  <div class="of-banner of-banner--info"><strong>Heads up.</strong> New report templates are available.</div>
  <div class="of-banner of-banner--success"><strong>Synced.</strong> All client records are up to date.</div>
  <div class="of-banner of-banner--warning"><strong>Reminder.</strong> Your billing profile is incomplete.</div>
  <div class="of-banner of-banner--danger"><strong>Action needed.</strong> Two integrations failed to connect.</div>
</div>

## Accessibility

- Use concise, plain language.
- Ensure text contrast passes WCAG AA.
- Provide links/actions when user intervention is required.
