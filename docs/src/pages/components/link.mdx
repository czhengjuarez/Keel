Use links for navigation and external references, not form actions.

## Example

<div class="demo demo__row">
  <a class="of-link" href="#">Read project notes</a>
  <a class="of-link" href="#">View invoice history</a>
  <a class="of-link" href="#" aria-disabled="true">Unavailable link</a>
</div>

## Accessibility

- Link text should make sense out of context.
- Avoid generic labels like “click here”.
- Preserve underlines on hover/focus.
